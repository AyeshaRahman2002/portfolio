Developed a healthcare application that utilizes machine learning and eye-tracking data to detect Autism Spectrum Disorder (ASD) in adults. The tool analyzes social interactions, facial expressions, and eye movement patterns using natural language processing (NLP). It provides user feedback based on questionnaires and interactive tasks, trained on synthetic data to improve accuracy.

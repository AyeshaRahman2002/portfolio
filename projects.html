Development and Implementation of a Digital Health AI Prototype for Identifying Autism Spectrum Disorder in Adults

This project developed an AI-based prototype to aid in the diagnosis of Autism Spectrum Disorder (ASD) in adults, addressing a gap in current diagnostic tools. Using machine learning models for eye-tracking analysis and note tracking, alongside Natural Language Processing (NLP) for social interaction simulations, the system provides users with insights into their ASD traits. The prototype integrates established diagnostic questionnaires (AQ, AQ-10, CAT-Q, RAADS-R) to offer a comprehensive tool for self-assessment, with the potential to guide users toward professional diagnosis.

Decision Tree Learning and Pruning Algorithms

This project focused on developing and evaluating decision tree models to predict outcomes based on provided datasets. Key aspects of the project included implementing a function to learn decision trees, testing the trees on training and test splits, and generating synthetic datasets to assess model performance as dataset size increased. An essential part of the project involved implementing pruning algorithms to prevent overfitting by removing nodes that did not significantly improve the model. The project also emphasized statistical evaluations, using metrics like error ratios and p-values to refine the decision tree's accuracy, providing a hands-on experience in machine learning with a focus on optimizing decision-making models.

xv6 Memory Management

In this project for COMP2211: Operating Systems, I implemented memory management functions in the xv6-riscv operating system, focusing on creating dynamic memory allocation with the malloc() and free() functions. The project involved using circular linked lists to efficiently manage memory allocation, minimizing fragmentation and waste. I also implemented the rqstMem() function to request additional memory when necessary, using system calls to expand the memory dynamically. This project enhanced my understanding of memory management in low-level systems programming and gave me practical experience in handling dynamic memory allocation in an operating system.
